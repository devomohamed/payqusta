# دليل المستخدم العادي لنظام PayQusta (عربي)

هذا الدليل مخصص للمستخدمين اليوميين (مدير الفرع، الكاشير، موظف المبيعات) لاستخدام النظام خطوة بخطوة.

## 1) تسجيل الدخول
1. افتح رابط النظام.
2. أدخل البريد الإلكتروني وكلمة المرور.
3. اضغط **تسجيل الدخول**.
4. لو نسيت كلمة المرور: اضغط **نسيت كلمة المرور** واتبع التعليمات.

**لقطة شاشة مقترحة:** شاشة تسجيل الدخول.

---

## 2) التعرف على الواجهة
- **الشريط الجانبي**: كل أقسام النظام (الرئيسية، بيع سريع، المنتجات، العملاء، الفواتير...).
- **الشريط العلوي**:
  - اسم الصفحة الحالية.
  - البحث السريع (يمكن استخدام `Ctrl + K`).
  - الإشعارات.
  - تغيير اللغة.
  - الوضع الليلي/النهاري.
- **زر تسجيل الخروج**: أسفل الشريط الجانبي.

**لقطة شاشة مقترحة:** الواجهة الرئيسية مع تمييز كل جزء.

---

## 3) أهم العمليات اليومية (Actions)

### 3.1 إضافة منتج جديد
1. من القائمة: **المنتجات > كل المنتجات**.
2. اضغط **إضافة منتج**.
3. أدخل: الاسم، السعر، الكمية، التصنيف.
4. اضغط **حفظ**.

**لقطة شاشة مقترحة:** نموذج إضافة منتج.

### 3.2 تعديل المخزون
1. من: **المنتجات > تسوية المخزون**.
2. اختر المنتج.
3. أدخل الكمية الجديدة أو فرق الزيادة/النقص.
4. احفظ التعديل.

**لقطة شاشة مقترحة:** شاشة تسوية المخزون.

### 3.3 بيع سريع (كاش)
1. من: **بيع سريع**.
2. اختر المنتج/امسح الباركود.
3. أدخل الكمية.
4. اختر طريقة الدفع.
5. اضغط **إتمام البيع**.

**لقطة شاشة مقترحة:** شاشة البيع السريع.

### 3.4 إضافة عميل
1. من: **العملاء**.
2. اضغط **عميل جديد**.
3. أدخل الاسم، الهاتف، العنوان.
4. احفظ.

**لقطة شاشة مقترحة:** نموذج إضافة عميل.

### 3.5 إنشاء فاتورة (كاش أو أقساط)
1. من: **الفواتير**.
2. اضغط **إنشاء فاتورة**.
3. اختر العميل والمنتجات.
4. اختر نوع الفاتورة:
   - **نقدي** أو
   - **أقساط** (تحديد المقدم، عدد الأقساط، التكرار).
5. اضغط **حفظ/إصدار الفاتورة**.

**لقطة شاشة مقترحة:** شاشة إنشاء فاتورة أقساط.

### 3.6 تسجيل دفعة قسط
1. افتح الفاتورة من قائمة الفواتير.
2. اضغط **تسجيل دفعة**.
3. أدخل قيمة الدفعة وتاريخها.
4. احفظ.
5. يمكن أيضًا **سداد كامل** من نفس الشاشة.

**لقطة شاشة مقترحة:** نافذة تسجيل الدفعة.

### 3.7 الموردين (شراء وسداد)
1. من: **الموردين**.
2. أضف مورد أو اختر مورد موجود.
3. سجل عملية شراء/توريد.
4. عند السداد اضغط **سداد** أو **سداد كامل**.

**لقطة شاشة مقترحة:** صفحة الموردين.

### 3.8 المصروفات
1. من: **المصروفات**.
2. اضغط **إضافة مصروف**.
3. أدخل النوع، القيمة، ملاحظات.
4. احفظ.

**لقطة شاشة مقترحة:** شاشة إضافة مصروف.

### 3.9 متابعة التقارير
- من: **التقارير**:
  - التقارير العامة.
  - التقارير التجارية.
  - أعمار الديون.
- استخدم الفلاتر (تاريخ/فرع/نوع) ثم **تصدير** إذا متاح.

**لقطة شاشة مقترحة:** شاشة التقارير بعد تطبيق فلتر.

---

## 4) الإشعارات والتنبيهات
- راجع جرس الإشعارات يوميًا.
- أهم التنبيهات:
  - أقساط مستحقة.
  - نقص مخزون.
  - رسائل دعم/مراجعات.
- افتح الإشعار مباشرة للانتقال للصفحة المطلوبة.

**لقطة شاشة مقترحة:** قائمة الإشعارات.

---

## 5) الإعدادات الأساسية للمستخدم
من **الإعدادات** يمكنك:
- تعديل بيانات الحساب.
- إعدادات المتجر.
- إعدادات التقسيط.
- إعدادات WhatsApp (لو مفعلة).
- إدارة المستخدمين والصلاحيات (للمدير فقط).

**لقطة شاشة مقترحة:** صفحة الإعدادات.

---

## 6) أخطاء شائعة وحل سريع
- **لا أستطيع الدخول**: تأكد من البريد/كلمة المرور، ثم استخدم “نسيت كلمة المرور”.
- **منتج لا يظهر**: راجع الصلاحيات أو الفرع الحالي أو حالة المنتج.
- **فاتورة لا تُحفظ**: تأكد أن العميل والمنتجات مكتملة وأن الكميات متاحة.
- **البيانات مختلفة بين الفروع**: تأكد من الفرع المختار أعلى الشاشة.
